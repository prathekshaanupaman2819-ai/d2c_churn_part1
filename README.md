# d2c_churn_part1
# Part 1: Data Audit and Exploratory Data Analysis

This repository contains the data audit and exploratory data analysis (EDA) for the D2C customer churn capstone project.

## Project Contents
* `eda_analysis.ipynb`: The jupyter notebook containing the full analysis and visualizations.
* `requirements.txt`: The list of libraries required to run this project.

## How to Run
1. Clone this repository to your local machine or environment.
2. Install the required libraries using `pip install -r requirements.txt`.
3. Open and run the `eda_analysis.ipynb` file.

## Academic Integrity
Analysis is grounded in the provided dataset and supported by visual evidence.
